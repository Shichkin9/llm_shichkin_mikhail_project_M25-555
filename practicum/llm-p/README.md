# Построение защищённого API для работы с большой языковой моделью (Шичкин Михаил)
# Цель работы
# Целью работы 
 - разработка серверного приложения на FastAPI, предоставляющего защищённый API для взаимодействия с большой языковой моделью (LLM) через сервис OpenRouter. 

В рамках задания были получены следующие навыки:
    - работа с FastAPI и асинхронным backend,
    - проектирование серверной архитектуры с разделением слоёв,
    - использование JWT для аутентификации,
    - интеграция внешних API (LLM),
    - работа с базой данных через SQLAlchemy,
    - управление зависимостями проекта через uv

# Запуск приложения:
uv run uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
